# Plan 03-05: Speed, Intensity, and Color Controls

## Objective
Build speed/intensity sliders and a 3-slot color picker using react-colorful. Users can tune effect parameters and set up to 3 simultaneous colors.

- @src/core/wled/WLEDControlService.ts — setSpeed, setIntensity, setColors
- @src/core/store/cubeStateStore.ts — speed, intensity, colors
- react-colorful — installed in 03-02
