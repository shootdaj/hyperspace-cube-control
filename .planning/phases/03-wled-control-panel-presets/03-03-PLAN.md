# Plan 03-03: Effect Browser

## Objective
Build a scrollable, searchable effect browser that fetches all 100+ WLED effects and lets the user activate one with a single tap. The active effect is highlighted and the 3D visualization updates via the state round-trip.

## Context
- @src/core/wled/WLEDRestClient.ts — getEffects() returns string[]
- @src/core/wled/WLEDControlService.ts — setEffect(index)
- @src/core/store/cubeStateStore.ts — effectIndex
- @test/mocks/virtualCube.ts — MOCK_EFFECTS array
