# Plan 05-04: Audio-to-LED Mapping Strategy

## Objective
Implement `AudioSpectrumMappingStrategy` that converts FFT spectrum data to 480 LED RGB values. Supports three visualization modes: spectrum (per-edge frequency bands with mel-scale grouping), energy (overall volume drives brightness), and waveform (amplitude spread across LEDs).

## Context
- @src/core/pipeline/types.ts — MappingStrategy interface, FrameData
- @.planning/phases/05-audio-reactive-input/05-RESEARCH.md — Frequency mapping research
- @src/stores/audioStore.ts — visualizationMode, sensitivity state

    Helper functions (private or module-level):
    - `hslToRgb(h: number, s: number, l: number): [number, number, number]` — standard HSL to RGB
    - `freqToMel(hz: number): number` — `2595 * Math.log10(1 + hz / 700)`

    - `buildMelBands(binCount: number, sampleRate: number, numBands: number): number[][]` — returns array of bin index groups for each mel band

    Class AudioSpectrumMappingStrategy implements MappingStrategy:
    - `readonly id = 'audio-spectrum'`
    - Private: `outputBuffer: Uint8Array`, `melBands: number[][]` (precomputed), `smoothedBands: Float32Array`
    - Constructor: allocate outputBuffer(480*3), precompute melBands for 1024 bins / 44100 Hz / 12 bands, allocate smoothedBands(12)
    - `map(frame: FrameData, ledCount: number): Uint8Array`:
      1. If no frame.spectrum, fill outputBuffer with 0, return it
      2. Read visualizationMode and sensitivity from audioStore
      3. Apply sensitivity threshold: zero out bins below sensitivity/255
      4. Switch on mode:
        - **spectrum**: Group bins into 12 mel bands, average amplitude per band, apply smoothing (alpha=0.3), for each edge (40 LEDs): hue = bandIndex/12, saturation = 1.0, lightness = amplitude * 0.5, write RGB to all 40 LEDs
        - **energy**: Average all bins for overall energy, find dominant frequency bin for hue, all 480 LEDs get same color with brightness = energy
        - **waveform**: Map bins linearly to 480 LEDs (interpolate if bin count != led count), hue from position (0..1 mapped to 0..300 degrees), lightness from amplitude
      5. Return outputBuffer

    Tests:
    - Zero spectrum → all black output
    - Spectrum mode with known input → verify edge colors differ by band
    - Energy mode → all LEDs same color
    - Waveform mode → LEDs vary across the strip
    - Sensitivity threshold → low values zeroed
    - hslToRgb: known conversions (red, green, blue, white)
    - melScale: verify 1000 Hz → ~1000 mel
