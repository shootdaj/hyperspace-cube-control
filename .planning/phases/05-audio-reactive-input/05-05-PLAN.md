# Plan 05-05: Audio Controls UI

## Objective
Build the AudioControls React component with device selector dropdown, start/stop button, gain slider, sensitivity slider, and visualization mode selector. Integrate as a new "Audio" tab in the ControlPanel.

- @src/components/ui/slider.tsx — shadcn slider component
- @src/components/ui/button.tsx — shadcn button component
