# Plan 05-01: Audio Device Enumeration & Store

## Objective
Create the Zustand audio store and implement audio device enumeration via `navigator.mediaDevices.enumerateDevices()`. Users can see all available audio input devices (mic, BlackHole, system audio) in a dropdown after granting permission.

## Context
- @src/core/pipeline/types.ts — InputPlugin, FrameData, MappingStrategy interfaces
- @src/stores/paintStore.ts — Reference Zustand store pattern
- @src/core/store/ledStateProxy.ts — Valtio proxy for hot-path data
- @.planning/phases/05-audio-reactive-input/05-RESEARCH.md — Research findings
